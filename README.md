# Windows 11 自动化装机与部署基建 (Win11-Init)

本工具箱提供了一套**工程化、模块化**的自动化脚本方案，专为需要将全新的 Windows 11 电脑配置为“**无头服务器**”（Headless Server）、**远程开发节点**或**常驻挂机设备**的场景所打造。

只需执行一个 `.bat` 即可一键完成网络环境穿透、账户越权管理、电源环境冻结、系统流氓限制清理以及服务器自动健康报告，秒级搭建一套可依赖的高负载生产力环境。

## 📁 核心文件结构

| 文件名 | 用途描述 |
| ------ | -------- |
| 🚀 `run_setup.bat` | **系统启动器**。自动请求管理员权限并运行部署脚本，捕获日志。**使用时始终双击运行该文件即可**。 |
| ⚙️ `setup_win11.ps1` | **核心业务流引擎**。包含多达 10 个核心业务模块和完备的健壮核查机制，负责执行所有的底层环境修改与系统服务重组。无交互式设计，可以静默到底。 |
| 📊 `dailyreport.ps1` | **自动化健康监控报告集**。与配置流联动。独立运行时可立即报告系统状态；配合 `setup_win11.ps1` 时会自动安装为常驻任务。 |

## ⚡ 实现的十大核心功能

脚本流将在执行中自动处理并输出以下 10 个环节的详细状态报告：

1. **原装 OpenSSH 安装与启用**：强制挂载系统组件源并激活后台 sshd 守护服务。
2. **SSH 连接极限优化**：打通 `Keep-Alive` 防止客户端无故离线，彻底将 SSH 登录进入的老旧交互终端强制转换为新版 PowerShell。
3. **上帝用户构建**：强制自动创建本地名 `user` (密码 `1234`，密码永不过期) 并硬性绑定进入 `Administrators` 顶权管理组。
4. **纯向全通防火墙**：摧毁安全屏障并放行 **“全协议全端口入站规则”**（极大方便纯内网的服务器间调用与外派内网穿透）。
5. **系统电源冻眠**：开启绝对高性能模板，强力锁定休眠规则，屏蔽笔记本插电池或者“合上盖子屏幕熄灭”引爆系统断网休眠的安全隐患。
6. **GUI 图形急救热线 (RDP)**：开启 Windows 内核层远程桌面并强通安全拦截，随时保障图形排障。
7. **深度防御瘫痪 (Defender)**：全方位关闭 Windows Defender 的实时防御，强制绕过误杀拦截，免疫运行开源及破解程序的阻隔。
8. **深度系统瘦身免打扰 (Debloat)**：斩断长达几十 GB 的睡眠大文件；镇压一切非必要 UAC 弹窗；**强制阻止 Windows Update 偷偷重启你的机器！**
9. **拥抱现代开源 (Chocolatey)**：植入包管理器。后期仅靠敲击 `choco install python git 7zip nodejs docker-desktop -y` 即可享受 Linux 体验般的光速开发环境部署。
10. **常驻 Gotify 系统健康监控**：在最后部署阶段自动将 `dailyreport.ps1` 深埋入系统并在每天的“偶数点”（每 2 小时）为你手机端的 Gotify 播报服务器最新的：IP地址、内存、CPU负载及 C盘余额。

## 📌 快速开始指南

1. **准备环境**：将本下载目录整个通过 U盘 或云盘打包拷贝至刚装好系统或者是目标 Windows 11 的桌面上。
2. **一键开跑**：无视任何环境配置，直接**双击运行 `run_setup.bat`**。
3. **同意授权**：在弹出的 Windows 管理大黄盾提示上点击**“是 / Yes”**，然后等待安装完成自动退出。
4. **完成与连入**：拔掉这台机器的键盘与显示器，返回你的 Mac/主力机，使用命令直冲进来：
   ```bash
   ssh user@<目标电脑的内网IP>
   # 密码默认：1234
   ```

## 🔍 附加排错：远程局域网共享探测 (命令行直连测试)

如果您想随手在任意另一台电脑上快速验证目标服务器（如 `192.168.0.7`）到底开放了哪些文件夹共享、网络是否能连通，请直接打开标准 `cmd` 终端，复制粘贴并执行下面这两句口诀：

```cmd
:: 1. 建立 IPC$ 基础认证空连接通道 (此时系统底层将获得该局域网 IP 的通行授权)
net use \\192.168.0.7\IPC$ "sm12345!" /user:유영

:: 2. 叩栓成功后，直接拉取展示该目标服务器对外暴露的纯净版共享列表
net view \\192.168.0.7
```

*(温馨提示：测试完毕后，您可以用 `net use \\192.168.0.7\IPC$ /delete` 随手拔除掉刚才建立的凭据，从而防止系统后续访问其它服务器时引发的通道冲突报错。)*

## ⚠️ 安全严重告警 (SECURITY WARNING)

该脚本是一把双刃剑：由于设计目标为“**纯生产及无头节点调试基线**”，其中包含了多项**放弃安全换取效率**的暴力操作（例如全放行防火墙，极弱硬广密码，强制关闭 UAC，强制瘫痪杀毒软件）。
**严禁将经过此脚本初始化的计算机在未设置任何路由硬件墙的前提下直接暴露公网（裸持公网 IP）**，否则将在数分钟之内遭遇恶意加密勒索等拦截。
