# Windows 11 自动化装机与部署基建 (Win11-Init v2.0)

本工具箱提供了一套**工程化、模块化**的自动化脚本方案，专为需要将全新的 Windows 11 电脑配置为“**无头服务器**”（Headless Server）、**远程开发节点**或**常驻挂机设备**的场景所打造。

**v2.0 优化版更新说明**：
- **更强的健壮性**：新增启动权限自动预检、模块化重入检测、环境变量实时刷新。
- **更深的监控度**：监控报告新增“高负载进程排行”及“多源 IP 冗余检测”。
- **透明化日志**：引入带毫秒级时间戳的全局日志系统，所有部署细节均可追溯。

## ⚡ 核心功能模块

脚本流将在执行中自动处理并输出以下环节的详细状态报告：

1. **OpenSSH 核心组件部署**：自动检测环境并静默挂载 OpenSSH-Server，确保 SSHD 服务自动启动并常驻。
2. **SSH 环境极限优化**：打通 `Keep-Alive` 链路，并将 SSH 默认终端强制转换为现代 PowerShell。
3. **上帝用户 (Headless Admin) 构建**：自动创建本地用户 `user` (密码 `1234`) 并硬性绑定至管理员组。
4. **全通防火墙策略**：放行 **“全协议全端口入站规则”**，为内网服务调用与穿透扫平障碍。
5. **电源环境冻结**：锁定高性能模式，屏蔽系统休眠、睡眠以及笔记本合盖导致的任务中断。
6. **RDP 急救通道**：开启 Windows 原生远程桌面 (Port 3389) 并自动放行防火墙策略。
7. **Defender 深度旁路**：全方位关闭实时防御与全盘扫描，允许开源或自定义工具无阻运行。
8. **系统减负与 UAC 静默**：清理睡眠大文件，镇压 UAC 弹窗，**并强制阻止系统自动更新引发的重启**。
9. **软件生态源 (Chocolatey)**：植入包管理器。后期仅需执行 `choco install python git 7zip -y` 即可快速部署环境。
10. **Gotify 2.0 监控体系**：自动安装定时任务，每 2 小时推送包含 **IP、CPU、内存、磁盘以及高负载进程列表** 的健康报表。

## 📌 快速开始指南

1. **环境准备**：将本下载目录拷贝至目标机器桌面。
2. **一键启动**：**以管理员身份双击运行 `run_setup.bat`**。
   - *提示*：默认开启调试模式（显示窗口）。如需后台静默运行，请右键编辑 `run_setup.bat` 将 `DEBUG_MODE` 修改为 `0`。
3. **查看进度**：脚本将自动弹出 PowerShell 窗口执行部署。完成后，详细报告将保存在 `setup_win11.log`。
4. **远程接入**：
   ```bash
   ssh user@<目标 IP>  # 默认密码: 1234
   ```

## ⚠️ 安全警告 (SECURITY WARNING)

该脚本设计目标为“**纯生产及无头节点调试基线**”，其中包含了多项**放弃安全换取效率**的暴力操作（例如全放行防火墙，极弱硬广密码，强制关闭 UAC，强制瘫痪杀毒软件）。
**严禁将经过此脚本初始化的计算机在未设置硬件防火墙的前提下直接暴露公网**，否则可能遭遇安全威胁。建议仅在受控的局域网环境或带有路由防火墙的环境下使用。
